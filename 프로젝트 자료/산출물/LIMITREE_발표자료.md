## 슬라이드 7: ERD (데이터베이스 설계)

```
┌────────────────────────────────────────────────────────────┐
│                    ERD (핵심 테이블)                       │
├────────────────────────────────────────────────────────────┤
│                                                            │
│  ┌──────────────┐                      ┌──────────────┐    │
│  │   USERINFO   │──────────────────────│    CART      │    │
│  │──────────────│         1:N          │──────────────│    │
│  │ userId (PK)  │──┐                   │ cartItemId   │    │
│  │ password     │  │                   │ productId(FK)│    │
│  │ name         │  │                   │ userId (FK)  │    │
│  │ email        │  │                   │ quantity     │    │
│  │ social       │  │                   │ color, size  │    │
│  └──────────────┘  │                   └──────────────┘    │
│         │          │                                       │
│         │          │  ┌──────────────┐ ┌──────────────┐    │
│    1:N  │          └──│  WISHLIST    │ │   COMPARE    │    │
│         │             │──────────────│ │──────────────│    │
│         │             │ wishlistId   │ │ compareItemId│    │
│         │             │ productId(FK)│ │ productId(FK)│    │
│         │             │ userId (FK)  │ │ userId (FK)  │    │
│         │             └──────────────┘ └──────────────┘    │
│         ▼                                                  │
│  ┌──────────────┐     ┌──────────────┐ ┌──────────────┐    │
│  │USERINFO_ROLES│     │   PRODUCT    │ │   CATEGORY   │    │
│  │──────────────│     │──────────────│ │──────────────│    │
│  │ role         │     │ id (PK)      │ │ categoryId   │    │
│  └──────────────┘     │ sku          │ │ nameKey      │    │
│                       │ name, price  │ │ parent (FK)  │←─┐ │
│                       │ discount     │ │ sortOrder    │──┘ │
│                       │ stock        │ └──────────────┘    │
│                       │ image        │    Self-Join        │
│                       └──────────────┘                     │
│                              │                             │
│                         1:N  │                             │
│                              ▼                             │
│  ┌──────────────┐     ┌────────────────┐                   │
│  │   ORDERS     │     │PRODUCT_VARIATION│                  │
│  │──────────────│     │────────────────│                   │
│  │ orderId (PK) │     │ id (PK)        │                   │
│  │ customerName │     │ productId (FK) │                   │
│  │ totalAmount  │     │ color          │                   │
│  │ orderStatus  │     │ image          │                   │
│  └──────────────┘     └────────────────┘                   │
│         │                    │                             │
│    1:N  │               1:N  │                             │
│         ▼                    ▼                             │
│  ┌──────────────┐     ┌────────────────────┐               │
│  │ ORDER_ITEMS  │     │PRODUCT_VARIATION_SIZE│              │
│  │──────────────│     │────────────────────│               │
│  │ orderItemId  │     │ id (PK)            │               │
│  │ orderId (FK) │     │ pVariationId (FK)  │               │
│  │ productId(FK)│     │ name (XS,S,M,L,XL) │               │
│  │ quantity     │     │ stock              │               │
│  │ price        │     └────────────────────┘               │
│  └──────────────┘                                          │
│                                                            │
│  ┌──────────────┐                                          │
│  │    REVIEW    │                                          │
│  │──────────────│                                          │
│  │ reviewId (PK)│                                          │
│  │ productId(FK)│                                          │
│  │ userId       │                                          │
│  │ rating       │                                          │
│  │ content      │                                          │
│  └──────────────┘                                          │
│                                                            │
└────────────────────────────────────────────────────────────┘
```
